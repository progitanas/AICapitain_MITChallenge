# AI Captain - Full Stack Frontend

## Architecture Professionnelle

Ceci est une **refonte complète du frontend** avec une UX premium sans aucun indicateur IA, conçue comme les grandes plateformes tech (Upwork, Vercel, etc.).

### Stack Technologique

- **React 18** avec TypeScript strict
- **Vite** pour le bundling (ultra rapide)
- **Tailwind CSS** pour le design système
- **React Router v6** pour la navigation
- **Zustand** pour la gestion d'état (léger & performant)
- **Axios** pour les appels API
- **Leaflet + React Leaflet** pour les cartes
- **Recharts** pour les graphiques
- **Lucide React** pour les icônes pro (SVG)
- **Framer Motion** pour les animations fluides

### Intégration Backend

Tous les appels API passent par `src/services/api.ts`:

```typescript
// Example
import { apiClient } from '@/services/api'

const result = await apiClient.post('/api/v1/route/optimize', {
  start_port: 'PORT_SG',
  end_port: 'PORT_HH',
  // ...
})
```

Le proxy Vite redirige `/api/*` vers `http://localhost:8000`.
